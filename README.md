# 公车采购审批管理系统

《公车采购审批管理系统》是一款针对公车购买审批管理的一款软件，它可以记录每辆车从申请、审批、购买、挂牌、报废、加油、维修、保险等管理的一系列情况。

具体功能包话如下：

车辆管理：购车申请、购车审批、车辆购买、牌号审批、车辆信息、牌号审批通过
车辆处置：报废申请、报废审批、报废查看

车辆编制：车辆编制

车辆费用：保险费用、维修费用、加油费用

软件特点如下：    
    1.界面设计简洁、美观、操作流程人性化，普通用户不需培训也能很快上手；   
    2.支持数据与execel的导出与导入。


本系统基于宏达数据库信息管理开发平台开发，下载后，双击文件夹中"dbimp.exe"文件即可完整运行对应的信息管理系统。
